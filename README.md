# Homebrew Tap for ax

macOS accessibility CLI that exposes the accessibility tree as JSON for AI agents.

## Installation

```bash
brew tap synthemesc/ax
brew install ax
```

## Usage

See [ax documentation](https://github.com/synthemesc/ax) for usage details.

## Permissions

After installation, you'll need to grant permissions:

1. **Accessibility**: System Settings > Privacy & Security > Accessibility
2. **Screen Recording** (for screenshots): System Settings > Privacy & Security > Screen Recording
